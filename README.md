# Goober Dash

2D multiplayer racing platformer for iPad and macOS.

## Overview

Fast-paced side-scrolling race where players compete across dynamically built tracks with physics hazards, power-ups, and LAN multiplayer.

## Features

- LAN multiplayer via Unity Netcode for GameObjects (up to 8 players)
- Two maps: Forest and Volcano, with distinct layouts and hazards
- In-game level editor with 39 block types and save/load
- AI opponents with 4 difficulty presets (Easy → Ultra)
- Physics: wall jumps, double jumps, dashes, ice surfaces, wind zones
- Full UI in Unity UI Toolkit (UXML/USS)

## Stack

Unity 2022 · C# · Netcode for GameObjects · Unity Transport (UDP) · New Input System · UI Toolkit

## Platforms

iPad (primary) · macOS
